Enterprise User Onboarding Automation Suite (PowerShell + WinForms)
Overview
This project is a full-featured Active Directory and Exchange Online onboarding automation tool built in PowerShell, complete with a graphical user interface (GUI) using Windows Forms. It streamlines the creation and configuration of new user accounts in an enterprise environment by automating:
- Active Directory user creation
- Attribute and extension attribute population
- Manager assignment
- Remote mailbox provisioning
- Mailbox validation
- Bulk onboarding from CSV
- Logging and audit tracking
The tool is designed to reduce manual onboarding time, eliminate repetitive tasks, and ensure consistent provisioning across the organization.

Features

User Creation
- Create new AD users with:
- First/Last name
- samAccountName (auto‑generation supported)
- Title
- Department
- Employee type
- Start date
- Template user cloning
- Secure password assignment

Attribute Management
- Automatically populate extension attributes
- Assign department, title, and address
- Set manager based on Employee ID lookup

Mailbox Provisioning
- Generate primary SMTP and remote routing addresses
- Create remote mailboxes
- Apply email address policies
- Validate mailbox creation

Bulk Onboarding
- Import a CSV file containing multiple users
- Automatically process each user through:
- AD creation
- Attribute assignment
- Manager assignment
- Mailbox provisioning
- Output results to an on‑screen log

Logging & Auditing
- All actions logged to a daily log file
- Built‑in log viewer tab
- Error handling with detailed messages

GUI Interface (WinForms)
- Multi‑tab layout:
- User Info
- Mailbox
- Bulk
- Logs
- Buttons enabled/disabled based on workflow state
- Real‑time output windows
- File picker for CSV import

Technologies Used
- PowerShell 5+
- ActiveDirectory Module
- Exchange Online Remote PowerShell
- Windows Forms (System.Windows.Forms)
- .NET Framework Classes
- CSV-based data ingestion

How It Works
1. User Info Tab
Enter user details → Create AD user → Apply attributes → Assign manager → Move to mailbox provisioning.
2. Mailbox Tab
Generate SMTP addresses → Create remote mailbox → Validate mailbox.
3. Bulk Tab
Select a CSV file → Run bulk onboarding → View results in real time.
4. Logs Tab
View the daily log file generated by the tool.

CSV Format Example
Your CSV should include fields similar to:
FirstName,LastName,samAccountName,Title,Department,EmployeeType,StartDate,Template,Password,ManagerID,StreetAddress
John,Doe,jdoe,Engineer,IT,FTE,2024-01-15,TemplateUser,TempPass123,12345,123 Main St


You can customize fields based on your environment.
